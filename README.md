# Horizon Evaluator AI

AI-based Horizon Europe proposal pre-evaluation platform.

## Features

- Upload PDF or DOCX proposal
- Enter Horizon Europe call information
- AI-based evaluation under:
  - Excellence
  - Impact
  - Quality and Efficiency of Implementation
- ESR-style comments
- Score dashboard
- Priority improvement actions

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
